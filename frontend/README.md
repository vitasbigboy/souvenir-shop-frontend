# Souvenir Shop Frontend

Frontend часть дипломного проекта интернет-магазина корпоративной сувенирной продукции.

## Технологии

- React
- TypeScript
- Vite

## Установка

Установить зависимости:

`npm install` 

Запустить проект:

`npm run dev` 

После запуска приложение будет доступно по адресу:

http://localhost:5173 

## Backend

Для корректной работы необходимо запустить Django Backend по адресу:

http://127.0.0.1:8000 

Frontend получает данные о товарах, категориях и заявках через REST API.

## Функционал

- Каталог товаров
- Карточка товара
- Корзина
- Оформление заявок
- Интеграция с Backend API
